EagleEye2
=========

Snapshot shodan results for quick sorting and evaluation.

System requirements (install with apt):
-------------------
 * xvfb
 * rabbitmq-server
 * redis-server(?)
 * chromium-browser

Manual downloads:
----------------
 * [chromedriver](http://code.google.com/p/chromedriver/downloads/list)

Python Requirements (install with pip):
-------------------
 * celery
 * selenium
 * pyvirtualdisplay
 * redis(?)
